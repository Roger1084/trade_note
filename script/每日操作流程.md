## 第一部分：盘前准备（8:00-9:00）

### 步骤1：大盘环境评估（手工 + 通达信）

**目的**：决定当天的整体仓位上限。

打开通达信上证指数日线图，查看：

| 条件 | 判断标准 | 结论 |
|------|---------|------|
| 上证指数 vs MA20 | 收盘在 MA20 上方 | 趋势偏多 |
| MA20 方向 | MA20 > REF(MA20, 5) | 20日线向上，趋势健康 |
| MA20 vs MA60 | MA20 > MA60 | 中期多头排列 |
| 近3天走势 | 连续收阳或阳多阴少 | 短期动能持续 |

**仓位决策规则：**

```
指数 > MA20 且 MA20向上 → 满仓上限 8 成
指数 > MA20 但 MA20走平 → 仓位上限 5 成
指数 < MA20 但 MA20向上 → 仓位上限 3 成（回调中谨慎）
指数 < MA20 且 MA20向下 → 仓位上限 1 成 或 空仓
```

> 大盘 20 日线可以用通达信直接画 MA20 在主图上观察。

### 步骤2：板块强弱扫描（手工 + 通达信）

**目的**：找到当天的主线板块，只在强势板块中选股。

**操作：**
1. 通达信输入 `.400` 回车 → 板块指数涨幅排名
2. 查看涨幅前 10 的行业板块
3. 对每个候选板块，查看其指数日线图：

| 确认项 | 标准 |
|--------|------|
| 板块指数 vs MA20 | 在 MA20 上方或刚站上 |
| 板块指数 MA20 方向 | 向上或走平后拐头 |
| 板块近 3 天涨幅 vs 大盘近 3 天 | 领先大盘（超额 > 0） |

4. 记录符合条件的板块名称（最多 3 个）：___________

> 进阶：用 Python 脚本（见附件）每天自动跑板块排名和强度筛选。

## 第三部分：买入执行

### 步骤5：精确买入点

**目的**：不在信号日追涨，等缩量回调后再动手。

**买入规则（按信号类型）：**

**A. 正常介入信号——等阴线缩量回调后买入**

信号出现当天不买。等次日或第三日：
- 出现缩量阴线（C < O，V < VMA5）
- 收盘不破 MA13
- 在收盘前 5 分钟（14:55）确认后买入

**B. 涨停后介入信号——等缩量回踩站上 MA13 后买入**

信号出现代表涨停后已经回调且站稳。当天尾盘买入。

**C. 强势横盘信号——可直接买**

横盘强势说明主力控盘强，信号当天尾盘买入。

**买入动作：**
1. 14:55-14:57 确认收盘形态
2. 确认大盘当天没有跳水
3. 以市价买入

### 步骤6：仓位计算

**目的**：用止损控制每笔最大亏损不超过总资金的 2%。

**计算公式：**

```
买入股数 = (总资金 × 2%) ÷ (买入价 - 止损价)
```

**止损价设定规则：**
1. 优先用 ATR 止损线（主图白色虚线 STOP_LINE）
2. 如果 ATR 止损线和现价距离 < 1%，用 MA13 作为止损
3. 买入价和止损线距离 > 4%，仓位减半

**示例：**
```
总资金 200,000
最大亏损 2% = 4,000
买入价 13.50，ATR 止损线 12.90
每股风险 = 13.50 - 12.90 = 0.60
可买股数 = 4,000 ÷ 0.60 = 6,600 股（约 66 手）
占用资金 = 66 × 100 × 13.50 = 89,100（占 44.5%）
```

> 总仓位不超过步骤 1 决定的上限。多只股票总风险加总不超过 6%。

## 第四部分：持仓管理

### 步骤7：每日持仓检查（收盘后，10 分钟）

**目的**：决定继续持有、减仓还是清仓。

对持仓中的每只股票，检查通达信主图上的信号：

**清仓条件——任一触发立即次日开盘卖：**

| 条件 | 信号 | 含义 |
|------|------|------|
| 收盘有效跌破 MA13 | 主图出现绿色"卖"标记 | 趋势失效 |
| 放量冲高回落 | 主图出现绿色"卖"标记 | 资金撤退 |
| 结构破坏 | 主图出现绿色"卖"标记 | 核心区被破 |

**减仓条件——出现则次日减半仓：**

| 条件 | 信号 | 含义 |
|------|------|------|
| 高位缩量滞涨 | 主图出现绿色"减"标记 | 动能衰减 |
| 连续多天不创新高 | 主图出现绿色"减"标记 | 攻击力下降 |

**继续持有条件——以下全部满足则不动：**

- 没有卖/减信号
- 收盘站在 MA5 上方
- 量能没有异常放大（V < VMA20 × 3）
- 均线开口没有明显收敛

### 步骤8：移动止损

**目的**：保护利润，不让盈利单变成亏损单。

**规则：**
1. 盈利超过 5% → 止损线上移到成本价（保本止损）
2. 盈利超过 10% → 止损线上移到盈利的 50% 位置
3. 盈利超过 20% → 止损线跟随 ATR 止损线上移

在通达信主图上，白色虚线自动跟随价格上移（ATR 止损线是动态计算的）。

"""
交易日志脚本
用途：记录每笔交易，输出统计分析
运行：python trade_log.py
"""
import csv
import os
from datetime import datetime

LOG_FILE = 'trade_log.csv'

def log_trade(date, code, name, action, price, shares, 
              entry_reason='', exit_reason='', 
              hold_days=0, mkt_pct=0, sector_pct=0, note=''):
    """记录一笔交易"""
    
    # 计算金额和盈亏
    amount = price * shares
    pnl_pct = 0  # 卖出时计算
    
    row = [
        date, code, name, action, price, shares, amount,
        entry_reason, exit_reason, hold_days, 
        f'{pnl_pct:.2f}%', f'{mkt_pct:.2f}%', f'{sector_pct:.2f}%', note
    ]
    
    file_exists = os.path.exists(LOG_FILE)
    with open(LOG_FILE, 'a', newline='', encoding='utf-8-sig') as f:
        writer = csv.writer(f)
        if not file_exists:
            writer.writerow([
                '日期', '代码', '名称', '操作', '价格', '股数', '金额',
                '买入理由', '卖出理由', '持仓天数', 
                '盈亏%', '大盘涨跌%', '行业涨跌%', '备注'
            ])
        writer.writerow(row)

if __name__ == '__main__':

### C. 仓位计算器脚本

```python
"""
仓位计算器
用途：根据止损距离自动计算可买股数
运行：python position_calc.py
"""

def calc_position(total_capital, entry_price, stop_price, max_risk_pct=0.02):
    """
    计算仓位和股数
    
    参数：
    - total_capital: 总资金
    - entry_price: 买入价
    - stop_price: 止损价
    - max_risk_pct: 单笔最大亏损比例（默认2%）
    
    返回：
    - shares: 可买股数（100的整数倍）
    - position_pct: 仓位占比
    - risk_per_share: 每股风险
    """
    max_loss = total_capital * max_risk_pct          # 最大亏损金额
    risk_per_share = entry_price - stop_price         # 每股风险
    
    if risk_per_share <= 0:
        raise ValueError("止损价必须低于买入价")
    
    shares = int(max_loss / risk_per_share / 100) * 100  # 取100的整数倍
    cost = shares * entry_price
    position_pct = cost / total_capital * 100
    
    # 如果风险距离过大，仓位减半
    if risk_per_share / entry_price > 0.04:
        shares = shares // 2
        cost = shares * entry_price
        position_pct = cost / total_capital * 100
        print("⚠️ 风险距离 > 4%，仓位已自动减半")
    
    print(f"总资金: {total_capital:,.0f}")
    print(f"买入价: {entry_price:.2f}")
    print(f"止损价: {stop_price:.2f}")
    print(f"每股风险: {risk_per_share:.2f} ({risk_per_share/entry_price*100:.1f}%)")
    print(f"最大亏损: {max_loss:,.0f}")
    print(f"建议买: {shares} 股 ({shares//100} 手)")
    print(f"占用资金: {cost:,.0f} ({position_pct:.1f}%)")
    
    return shares, position_pct

if __name__ == '__main__':
    calc_position(
        total_capital=200000,
        entry_price=13.50,
        stop_price=12.90,
        max_risk_pct=0.02
    )
```

## 关键原则提醒

1. **没信号不出手**——错过比做错强
2. **大盘不行减仓到 3 成以下**——不和市场对抗
3. **缩量阴线是买点不是卖点**——上升趋势中的缩量回调是最好的上车机会
4. **破了 MA13 就清仓**——趋势破了没有任何理由留
5. **亏损不加仓**——补仓是把错误放大
6. **盈利加仓只在趋势完好时**——涨 5%+ 且信号未消失才考虑
7. **单只股票不超过 30%**——分仓就是风控
8. **每天写日志**——不写日志的交易者没有进步
